# Factors Affecting Crime in Australia

A research project investigating the socioeconomic, demographic, and environmental factors that influence crime rates across Australia.

## Project Overview

This project analyses crime data from Australian states and territories to identify key drivers of criminal activity. It draws on publicly available datasets from the Australian Bureau of Statistics (ABS), state police services, and other government sources.

## Research Questions

- What socioeconomic factors (e.g. unemployment, income inequality, poverty) are most strongly associated with crime rates?
- How do demographic variables (e.g. age distribution, population density, urbanisation) correlate with different crime types?
- Are there regional or state-level patterns in crime that cannot be explained by socioeconomic factors alone?

## Data Sources

- [Australian Bureau of Statistics — Crime and Justice](https://www.abs.gov.au/statistics/people/crime-and-justice)
- State and territory police open data portals
- ABS Census data (socioeconomic and demographic variables)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/<your-username>/australia-crime-factors.git
   cd australia-crime-factors
   ```
2. Install dependencies (if using Python):
   ```bash
   pip install -r requirements.txt
   ```
3. Open notebooks in `notebooks/` to explore the analysis.
